## Installation and Setup

```bash
# Clone the repository
git clone https://github.com/vickynina006/My-Portfolio-

# Navigate into the project folder
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
# React + Vite
```
